What you get

One window, many sessions

Tabs per working directory. Drag to reorder, double-click to rename.

Live state per tab

Running · idle · awaiting-user · awaiting-permission — driven by Claude Code's real hooks, not screen scraping.

Click to focus

OS notification or tray icon pops; clicking lands you on the exact tab needing attention.

Activity-aware tree

Per-session file tree that lights up as Claude reads, writes, edits, or deletes.

Command palette & search

Ctrl+K to jump anywhere. Ctrl+F to grep the terminal scrollback.

Undo destructive actions

Closed a session by mistake? 5-second toast lets you reopen at the same cwd.
